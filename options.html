Welcome to RAIN

Welcome to RAIN, the Right-click Analyst's IOC Navigator. Named after a particularly relentless miniature dachshund, it is a dedicated SOC tool designed to speed up the OSINT hunting of IOCs. Built to eliminate the friction of manual copypasting and excessive tab switching, RAIN bridges the gap between web text and actionable threat intelligence.

Simply highlight an IOC on any webpage and open your right-click context menu and RAIN automatically cleans the string and empowers you to immediately launch investigations.

Automatic Refanging & Cleaning

RAIN automatically sanitizes and refangs highlighted indicators on the fly before executing search queries. Supported refanging features include:

  • Protocol Refanging: Automatically converts defanged protocols like hxxps://, hxxp://, meows://, or meow:// back to standard https:// or http://.
  • Obfuscated Dots & Colons: Replaces bracketed, parenthesized, or verbal representations of periods and colons (e.g., [.], (.), {.}, [dot], (dot), [:], and (:)) with actual periods (.) and colons (:).
  • Surrounding Trimming: Strips away accidental leading or trailing quotes, parentheses, brackets, or angle brackets (e.g., ", ', [, ], (, ), <, >) highlighted along with the IOC.

Variables & Placeholders

Variables replaced dynamically by RAIN during queries.

IP_VALUE Replaces with the extracted IPv4 or IPv6 address.
DOMAIN_VALUE Replaces with the extracted domain name or URL.
HASH_VALUE Replaces with the extracted MD5, SHA-1, or SHA-256 file hash.
ACCOUNT_VALUE Replaces with the extracted user account name, email address, or UPN.
CVE_VALUE Replaces with the extracted CVE vulnerability identifier (e.g., CVE-YYYY-NNNN).
